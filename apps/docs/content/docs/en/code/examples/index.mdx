# Examples

These examples use A3S Code v3.1.0 concepts: ACL configuration, explicit environment-variable credentials, session APIs, task-based delegation, automatic subagent delegation, `.a3s/agents`, AHP, direct tools, verification, and optional queue infrastructure.

Start with [Quick Start](/docs/code/examples/quick-start), then move to streaming, model switching, hooks, security, and release-style verification.
