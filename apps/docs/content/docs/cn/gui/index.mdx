# A3S GUI

`a3s-gui` 是 A3S 应用的原生 UI renderer 原型。它接受 React-Aria 形态的语义 UI 树，
并把它们降成紧凑的 native UI IR，再交给平台 adapter 渲染，无需 WebView。

<Callout type="warn">
在当前 monorepo 工作区中，`crates/gui` 存在，但未记录在 `.gitmodules` 中。这个页面
描述的是当前本地项目形态，不应被理解为已发布 submodule 契约。
</Callout>

## 能力地图

| 领域 | 当前能力 |
| --- | --- |
| Authoring model | Web 兼容的 React/TSX 形态 tree，支持 React Aria 语义、DOM 风格 props、inline style、aria/data attribute 和命名事件回调。 |
| Native IR | Accessibility-first tree，包含 native role、state、label、value、style token、event 和 children。 |
| Runtime | `GuiRuntime`、keyed renderer diff engine、action registration、event routing，以及用于验证的 headless recording backend。 |
| Command stream | 可序列化 native command，例如 create、update、insert child、remove、set root。 |
| Platform planning | AppKit、WinUI、GTK planning adapter 将 IR 映射为 native widget blueprint。 |
| Native paths | crate manifest 中存在 feature-gated AppKit、WinUI、GTK native adapter paths。 |
| TypeScript bridge | `sdk/typescript` 下有原型 TypeScript SDK，提供 JSX/runtime protocol helper。 |

## 边界

A3S GUI 和 A3S WebView 不同。WebView 在嵌入式浏览器中渲染 web 内容；GUI 的目标是把
Web 兼容的语义 UI 直接渲染成原生控件。
